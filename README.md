# React Components Lab – Prof Platform

Ce projet est un **TP React** qui présente les bases du développement avec **React.js** en utilisant plusieurs composants simples.

Le projet démontre :
- La création de composants React
- L'utilisation des props
- L'affichage dynamique des données
- L'organisation d'une application React simple

---

# Technologies utilisées

- React.js
- JavaScript (ES6)
- HTML / CSS
- Node.js
- npm

---

# Lancer le projet

```bash
npm start
```

Puis ouvrir dans le navigateur :

```
http://localhost:3000
```

---

# Fonctionnalités du projet

Le projet contient plusieurs composants React :

### HelloWorld
Affiche un message simple.

### Greeting
Affiche un message de salutation avec props.

### Profil
Affiche les informations d’un utilisateur.

### ListeCourses
Affiche une liste d’éléments.

### Voiture
Affiche les informations d’une voiture.
